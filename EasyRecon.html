Welcome to "EasyRecon"

When performing passive recon on a target, there are dozens of tools we can use to gather various pieces of intel on our target. This tool will allow us to parse these utilities easily. For more details on the included resources and others, check out the following pages of my Gitbook:

S0cm0nkey's Security Reference Guide: Google Dorks

**Note: You must allow popups from this page for this tool to work correctly**

Blue: Single Tools, Green: Launches Multiple Tools, Red: Requires Manual Input, Yellow: Manual Input Multiple Objects


Recon Google Dorks: These are popular google dorks that can revieal interesting pages of your target domain.




Recon Sub-Domains: These are popular google dorks and osint tools that can revieal sub-domains of your target domain.




Recon Referenced Content: These are popular webpages that contain references to your target domain.




Cyber Search Engines and Internet Scanners: These are tools can be used to return miscellaneous data for the target .
